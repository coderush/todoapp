# todoapp

A simple todo application which can be used for bootstrapping a project.

- Frontend stack : 
  - backbone.js
  - require.js
  - less
  - flat-ui
  - bootstrap (v3.2.0) 
  - jquery (v2.1.1)
  - dustjs (v2.4.2)
- Backend stack : 
  - mongoose (3.8.17)
  - express (4.9.5)
  - krakenjs (1.0.3)
